Support site 1.2.0 13 July 2026

  • Audited product instructions and quoted labels against the Redline 1.0 interface and implementation.
  • Corrected the sidebar path to Page → Save changes and distinguished it from the right-click Save source command.
  • Added detailed Capture & annotations, Dossiers & tags, and Library guides.
  • Documented the exact Global Search, JSON import, duplicate-strategy, export-scope, Data Health, and citation-refresh behavior.
  • Replaced older illustrative product views with captures from the current built extension and clearly labeled sample data.
  • Added a dependency-free validator for local links, anchors, documentation navigation, search targets, and sitemap/SEO coverage.

Support site 1.1.0 13 July 2026

  • Added local documentation search with no hosted index or submitted queries.
  • Added a browser-support and known-limitations guide.
  • Added a local-only environment summary builder for public-safe issue context.
  • Added public release notes and expanded cross-links between troubleshooting, privacy, and support.
  • Kept the site dependency-free and compatible with GitHub project-page paths.

Support site 1.0.0 13 July 2026

  • Published the Redline product homepage and representative interface showcase.
  • Added getting-started, core workflow, backup/export, troubleshooting, FAQ, privacy, and support pages.
  • Added privacy-aware GitHub issue forms and private vulnerability-reporting guidance.
  • Added direct GitHub Pages publishing from main and a closed-source repository notice.

Redline 1.0.0

  • Initial packaged release for desktop Chrome-family browsers and Firefox.
  • Added local page, linked-source, image-reference, and selected-text capture.
  • Added typed annotations, notes, tags, dossiers, reading status, and importance.
  • Added local Library search, highlight restoration, JSON import, and all-data JSON backup.
  • Added CSV ZIP, XML, Markdown, BibTeX, RIS, CSL JSON, EndNote XML, academic Markdown, and bibliography exports.
  • Added local citation metadata extraction and editing.

Questions about a change?

Check known limitations and troubleshooting first. If behavior changed unexpectedly, use the support page to prepare a reproducible, non-sensitive report.
