Two version tracksRedline application versions and public-support-site versions are listed separately. A documentation update does not imply that the installed extension changed.

Support site 1.6.0 15 July 2026

  • Added a streamlined release selector to the top of every documentation table of contents and the documentation overview.
  • Published complete, searchable documentation snapshots for Redline 1.0.0 and 1.0.1 while keeping the stable /docs/ routes on the latest 1.0.2 guidance.
  • Added safe update guidance: browser updates should be automatic; check the marketplace when behind; export All data before reinstalling.

Redline 1.0.2 15 July 2026

  • Added the local Evidence Compiler for dossier claims, six exact annotation-to-claim relationship types, reviewable conclusions, and grouped supporting, challenging, and contextual passages.
  • Added deterministic evidence-gap audits without source-credibility or truth scoring.
  • Added claim search plus claim and evidence-relation coverage in JSON backup, CSV, XML, Markdown, and focused export scopes.
  • Improved tag and dossier assignment controls, including suggestions from the existing tag vocabulary.

Support site 1.5.0 15 July 2026

  • Added a dedicated Evidence Compiler guide and wired it into every documentation sidebar, the docs index, local search, and sitemap.
  • Updated homepage, Library, capture, core-concept, backup/export, troubleshooting, FAQ, privacy, and support guidance for claims and evidence relations.
  • Documented deterministic audit limits and evidence-aware JSON, CSV, XML, Markdown, and scoped exports.

Support site 1.4.1 14 July 2026

  • Fixed documentation-search navigation so section results retain and restore the promised heading fragment.
  • Re-resolved section links after destination-page loading and added consistent heading scroll spacing.
  • Updated the public release notes and Library guide for Redline 1.0.1.

Support site 1.4.0 14 July 2026

  • Expanded the browser-bookmark migration guide to cover Chrome-family bookmarks HTML and Firefox Bookmark Backup JSON.
  • Updated the homepage, FAQ, troubleshooting, privacy explanation, and documentation search for browser-neutral onboarding.
  • Clarified the difference between browser-bookmark migration files and the Redline All data JSON used for full vault-record recovery.

Support site 1.3.0 14 July 2026

  • Added the Chrome-family bookmarks HTML migration guide and browser-specific availability notes.
  • Documented how bookmark URLs, titles, dates, duplicate URLs, unsupported schemes, and full folder paths map into Redline.
  • Clarified that JSON remains the only full-fidelity Redline vault backup and restore format.
  • Added bookmark migration to local documentation search, the FAQ, and the public product overview.

Support site 1.2.0 13 July 2026

  • Audited product instructions and quoted labels against the Redline 1.0 interface and implementation.
  • Corrected the sidebar path to Page → Save changes and distinguished it from the right-click Save source command.
  • Added detailed Capture & annotations, Dossiers & tags, and Library guides.
  • Documented the exact Global Search, JSON import, duplicate-strategy, export-scope, Data Health, and citation-refresh behavior.
  • Replaced older illustrative product views with captures from the current built extension and clearly labeled sample data.
  • Added a dependency-free validator for local links, anchors, documentation navigation, search targets, and sitemap/SEO coverage.

Support site 1.1.0 13 July 2026

  • Added local documentation search with no hosted index or submitted queries.
  • Added a browser-support and known-limitations guide.
  • Added a local-only environment summary builder for public-safe issue context.
  • Added public release notes and expanded cross-links between troubleshooting, privacy, and support.
  • Kept the site dependency-free and compatible with GitHub project-page paths.

Support site 1.0.0 13 July 2026

  • Published the Redline product homepage and representative interface showcase.
  • Added getting-started, core workflow, backup/export, troubleshooting, FAQ, privacy, and support pages.
  • Added privacy-aware GitHub issue forms and private vulnerability-reporting guidance.
  • Added direct GitHub Pages publishing from main and a closed-source repository notice.

Redline 1.0.1 14 July 2026

  • Made browser-bookmark migration available in both Chrome and Firefox builds.
  • Added Firefox Bookmark Backup JSON alongside Chrome-family bookmarks HTML, with clearer local previews, duplicate handling, unsupported-URL reporting, full folder-path dossiers, and safer favicon handling.
  • Added Library activity and health summaries, pagination for larger page and dossier collections, and clearer page and dossier detail sections.
  • Kept browser-bookmark migration separate from Redline All data JSON backup and restore.

Redline 1.0.0

  • Initial packaged release for desktop Chrome-family browsers and Firefox.
  • Added local page, linked-source, image-reference, and selected-text capture.
  • Added typed annotations, notes, tags, dossiers, reading status, and importance.
  • Added local Library search, highlight restoration, JSON import, and all-data JSON backup.
  • Added Chrome-only Chrome/Netscape bookmarks HTML import with local preview and folder-to-dossier migration.
  • Added CSV ZIP, XML, Markdown, BibTeX, RIS, CSL JSON, EndNote XML, academic Markdown, and bibliography exports.
  • Added local citation metadata extraction and editing.

Questions about a change?

Check known limitations and troubleshooting first. If behavior changed unexpectedly, use the support page to prepare a reproducible, non-sensitive report.
